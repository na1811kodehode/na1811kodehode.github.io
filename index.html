Introduction

Python is a high-level, versatile programming language known for its simplicity and readability. It is widely used in web development, data analysis, artificial intelligence, automation, and more. Python emphasizes code clarity, allowing developers to write less code to achieve more.

Key features of Python:

  • Easy to learn and use for beginners.
  • Cross-platform compatibility.
  • An extensive standard library ("batteries included") for various tasks, such as file handling, data manipulation, and networking.

Python can be extended for specialized purposes through external libraries like Django for web development, TensorFlow for machine learning, and pandas for data analysis.

Variables and Data Types

Variables in Python are dynamically typed, meaning you don't need to declare their types explicitly. Simply assign a value, and Python determines the type automatically.


x = 10        # Integer
y = 3.14      # Float
name = "John" # String
Rules for naming variables:
  • Must begin with a letter or underscore (_).
  • Can contain letters, digits, and underscores but cannot start with a digit.
  • Python is case-sensitive (Name and name are different).

Data Types

Python supports several data types:
  1. Primitive Types:
    • int: Whole numbers (e.g., 42).
    • float: Decimal numbers (e.g., 3.14).
    • str: Strings of text (e.g., ).
    • bool: True/False values (True, False).
  2. Composite Types:
    • list: Mutable ordered collections (e.g., [1, 2, 3]).
    • tuple: Immutable ordered collections (e.g., (1, 2, 3)).
    • dict: Key-value pairs (e.g., {"key": "value"}).
    • set: Unordered collections of unique elements (e.g., {1, 2, 3}).

Control Flow

If Statements Python uses indentation to define blocks of code. An if statement evaluates a condition and executes code accordingly:

x = 10
if x > 5:
    print("x is greater than 5")
else:
    print("x is 5 or less")

elif allows multiple conditions:

if x > 10:
    print("x is greater than 10")
elif x == 10:
    print("x is exactly 10")
else:
    print("x is less than 10")

Loops

  1. While Loop: Executes as long as the condition is true: 
    n = 0
while n < 3:
    print(n)
    n += 1
  2. For Loop: Iterates over a sequence or range of values: 
    for i in range(3):
    print(i)
Functions

Functions in Python are defined using the def keyword. They allow you to encapsulate code for reusability and clarity.

def greet(name):
    return f"Hello, {name}!"

Call the function:

print(greet("World"))  # Output: Hello, World!

Functions can include:

  • Positional arguments: 
    def add(a, b):
    return a + b
print(add(5, 3))  # Output: 8
  • Default values: 
    def add(a, b=0):
    return a + b
print(add(5))  # Output: 5
  • Keyword arguments: 
    print(add(b=3, a=5))  # Output: 8
Constants and Reference

Constants in Python are not strictly enforced but are represented by naming variables in uppercase by convention:

PI = 3.14159
MAX_USERS = 100

Although these are treated like variables, developers typically refrain from modifying their values.
